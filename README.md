# Fifteen Puzzle Using Golang
`go-fifteen-puzzle` is a [Fifteen Puzzle](https://en.wikipedia.org/wiki/Fifteen_puzzle) using golang and ncurses.

# Install
go get -u github.com/yaozongyou/go-fifteen-puzzle
